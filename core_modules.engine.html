core_modules.engine.db_engine(server, db_name, db_user, db_pass)[fuente]

Crea un motor de conexión SQLAlchemy para una base de datos SQL Server.

Parámetros:

  • server (str) – Nombre o dirección del servidor SQL.

  • db_name (str) – Nombre de la base de datos.

  • db_user (str) – Usuario de la base de datos.

  • db_pass (str) – Contraseña del usuario.

Devuelve:

Objeto de conexión SQLAlchemy.

Tipo del valor devuelto:

sqlalchemy.engine.Engine

core_modules.engine.pyodbc_execute_commad(pyodbc_engine, command)[fuente]

Ejecuta un comando SQL usando pyodbc (INSERT, UPDATE, DELETE, etc.).

Establece conexión con la base de datos, ejecuta el comando y realiza commit. En caso de error, imprime la excepción y hace rollback.

Parámetros:

  • pyodbc_engine (str) – Cadena de conexión pyodbc.

  • command (str) – Comando SQL a ejecutar.

Devuelve:

None

core_modules.engine.query_df(db_user, db_pass, sql_query, server, db, params={}, dtype={})[fuente]

Ejecuta una consulta SQL y devuelve los resultados en un DataFrame.

Permite el uso de parámetros (params) y tipos de datos (dtype) opcionales. En caso de error en la conexión o consulta, devuelve 0.

Parámetros:

  • db_user (str) – Usuario de la base de datos.

  • db_pass (str) – Contraseña del usuario.

  • sql_query (str) – Consulta SQL a ejecutar.

  • server (str) – Nombre o IP del servidor SQL.

  • db (str) – Nombre de la base de datos.

  • params (dict, optional) – Parámetros para pasar a la consulta. Default es {}.

  • dtype (dict, optional) – Tipos de datos a forzar en las columnas. Default es {}.

Devuelve:

Un DataFrame con los resultados o 0 en caso de error.

Tipo del valor devuelto:

pd.DataFrame | int
