core_modules.directories.route_config(config)[fuente]

Devuelve la ruta completa de un archivo de configuración (carpeta “configs”).

Adapta la ruta base según el entorno de ejecución (script, ejecutable o Jupyter).

Parámetros:

config (str) – Nombre del archivo de configuración (incluyendo extensión).

Devuelve:

Ruta completa del archivo en la carpeta configs.

Tipo del valor devuelto:

str

core_modules.directories.route_sql(proceso, pais, query_type)[fuente]

Devuelve la ruta completa de un archivo SQL, considerando el entorno de ejecución.

La función adapta la ruta base dependiendo de si el código está siendo ejecutado como un ejecutable (PyInstaller), como script .py, o en un entorno Jupyter.

Parámetros:

  • proceso (str) – Nombre del proceso o subcarpeta dentro de “queries”.

  • pais (str) – Nombre del país (se espera un archivo <pais>.sql).

  • query_type (str) – Tipo de consulta (por ejemplo: “raw”, “final”, etc.).

Devuelve:

Ruta completa del archivo .sql.

Tipo del valor devuelto:

str
