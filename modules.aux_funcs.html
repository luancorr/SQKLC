modules.aux_funcs.PU_PCU(df)[fuente]

Calcula las columnas PU (Precio Unitario) y PCU (Precio por Cantidad Unitaria) para los actos.

La lógica varía según el tipo de compra (columna “Forma_Compra”) y otras columnas clave como “Preco_Unitario”, “Quantidade”, “Coef1_VW”, “UniPack” y “FlgGranel_VW”. Si algunas colunas clave no están disponibles, se renombran temporalmente para asegurar compatibilidad.

Parámetros:

df (pd.DataFrame) – DataFrame con columnas relacionadas a precios y empaques.

Devuelve:

DataFrame con las columnas “PU” y “PCU” calculadas y agregadas.

pd.DataFrame

modules.aux_funcs.first_digit(x)[fuente]

Extrae el último dígito significativo de un número, con lógica especial para negativos, ceros y valores iniciados por 7.

Esta función se utiliza para construir ordenamientos personalizados de agrupamientos basados en el campo “Valid” u otros criterios numéricos.

Reglas especiales: - Si comienza con “7” → retorna 0. - Si es negativo → retorna 99 seguido del primer dígito. - Se zeros à esquerda → retorna 990. - Caso padrão → retorna el último dígito.

Parámetros:

x (int | float) – Valor numérico a analizar.

Devuelve:

Dígito transformado para ordenación.

Tipo del valor devuelto:

int
