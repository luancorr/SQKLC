core_modules.sqlquery module

core_modules.sqlquery.sql_query(url)[fuente]

Carga y convierte un archivo .sql en un objeto ejecutable de SQLAlchemy.

Lee el contenido de un archivo SQL plano y lo convierte en un objeto sqlalchemy.text que puede ser utilizado em consultas SQL parametrizadas.

Parámetros:

url (str) – Ruta del archivo .sql.

Devuelve:

Objeto SQL listo para ejecutar con SQLAlchemy.

Tipo del valor devuelto:

sqlalchemy.sql.elements.TextClause
